## ğŸ¯ Design Specifications

### Typography
- **Display Font**: Cal Sans, Satoshi, or similar elegant display font
- **Body Font**: Inter, -apple-system fallback
- **Heading Sizes**: 72px hero, 44px sections, 20px cards
- **Letter Spacing**: -0.03em for large headings, -0.01em for body

### Color Palette
```
Background: #FAFAFA (off-white)
Text Primary: #171717 (charcoal)
Text Muted: #737373 (gray)
Border: #E5E5E5 (light gray)
Accent: #2563EB (blue)
Accent Hover: #1D4ED8 (darker blue)
White: #FFFFFF
```

### Spacing
- Section Padding: 128px vertical
- Card Padding: 40px
- Grid Gap: 32px
- Button Padding: 14px 28px
- Border Radius: 8px buttons, 12px cards

### Shadows
- Subtle: 0 1px 3px rgba(0,0,0,0.06)
- Hover: 0 8px 16px rgba(0,0,0,0.12)
- No heavy shadows, keep minimal
